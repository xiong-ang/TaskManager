# Task Manager      
> 实现多线程Task任务管理，提供了一个统一框架，可以扩展           

## Interface      
* MaxTasks           
设置可以同时运行的最大Task数         
* Register    
注册Task        
* Response            
进行用户输入              
* CancelAllTask          
取消所有Task              
* Release      
清理TaskManager
